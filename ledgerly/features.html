Features

Everything you need for treasury management. Nothing you don't.

Core Functionality

Multiple Expense Sheets

Organize your finances across different categories. Create separate sheets for actual expenses, planned budgets, specific trips, or any custom categorization you need.

  • Actual-to-date tracking
  • Trip-specific budgets
  • Expected expenses planning
  • Custom sheet creation

Transaction Management

Add income and expenses with clear visual indicators. Real-time validation ensures your data is always accurate.

  • Color-coded transaction types (green for income, red for expenses)
  • Auto-filled dates and running balances
  • Optional categories and notes
  • Transaction history with last 20 entries

Privacy & Security

100% Local Storage

Your financial data never leaves your computer. No cloud sync, no external servers, no data collection. Complete privacy guaranteed.

Excel-Based Storage

Data is stored in standard Excel files that you control. Place them anywhere—your desktop, OneDrive, or external drive. Easy to backup and migrate.

No External Connections

Works completely offline. No API calls, no telemetry, no tracking. Your financial records are yours alone.

Error Prevention

Transaction Type Validation

Clear visual distinction between income and expenses prevents costly mistakes. Large confirmation dialogs show all details before committing.

Amount Validation

Real-time validation as you type. Decimal place checking, warnings for unusually large amounts, and prevention of negative values.

Data Validation

Date format checking (YYYY-MM-DD), required field validation, and Excel file lock detection ensure data integrity.

User Confirmation

Every transaction requires confirmation. Review type, sheet, amount, and description before saving. Easy to catch mistakes.

Visualizations & Reports

Expense Breakdown

Interactive pie chart showing spending by category. Hover to see exact amounts. Instantly identify where your money goes.

Balance Over Time

Line chart visualization of your balance history. Track trends and see how your finances evolve month over month.

Monthly Summary

Grouped bar chart comparing income versus expenses by month. Quickly identify high-spending periods.

Markdown Export

Generate professional markdown reports organized by month with summary totals. Perfect for sharing with board members.

User Experience

Modern Interface

Built with CustomTkinter for a clean, modern look. Dark theme that's easy on the eyes during long sessions.

Visual Indicators

Balance displayed prominently—green for positive, red for negative. Color-coded buttons and clear transaction types.

Sheet Switching

Dropdown selector to quickly switch between expense sheets. Each sheet maintains its own transactions, balance, and history.
