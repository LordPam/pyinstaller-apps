Installation Instructions

  1. Windows: Download the .exe file and run it. You may need to click "More info" and "Run anyway" if Windows SmartScreen appears.
  2. macOS: Download the .app file, move it to Applications, and right-click → Open the first time (to bypass Gatekeeper).
  3. Linux: Download the binary, make it executable with chmod +x ledgerly, then run ./ledgerly

System Requirements

  • Operating System: Windows 10+, macOS 10.14+, or Linux
  • Memory: 512 MB RAM minimum
  • Storage: 100 MB free space
  • No internet connection required (runs completely offline)

Key Features

Core Functionality

  • Multiple Expense Sheets: Manage separate sheets for different purposes (actual-to-date, ocean-diver, egypt, weekend-trips, expected)
  • Transaction Management: Add income and expenses with clear visual distinction and real-time validation
  • Running Balance: Automatic balance calculations with visual indicators (green for positive, red for negative)
  • Transaction History: View recent transactions with full details

Data Security

  • 100% Local Storage: No cloud, no external connections
  • Excel-Based: Store data in Excel files on your OneDrive (or anywhere you choose)
  • No External API Calls: All data stays on your machine
  • Complete Privacy: No telemetry, no tracking, no data collection

Error Prevention

  • Transaction Type Validation: Clear visual distinction between Income (green) and Expense (red)
  • Amount Validation: Real-time validation with decimal place checking and warnings for unusual amounts
  • Data Validation: Date format checking, required field validation, and file lock detection
  • User Confirmation: Every transaction requires confirmation before saving

Visualization & Reports

  • Expense Breakdown: Interactive pie chart showing spending by category
  • Balance Over Time: Line chart showing how balance changes
  • Monthly Summary: Bar chart comparing income vs expenses by month
  • Markdown Export: Generate professional markdown tables grouped by month

How to Use Ledgerly

First-Time Setup

  1. Launch the application
  2. Select an existing Excel file or create a new one
  3. The app will automatically create all required sheets
  4. Start adding transactions!

Adding Transactions

  1. Select transaction type: EXPENSE (red) or INCOME (green)
  2. Fill in details: Date (auto-filled), Description, Amount, Category (optional), Notes (optional)
  3. Click "Add Transaction" and review the confirmation dialog
  4. Confirm to save the transaction

Best Practices

  • Enter transactions regularly to avoid backlog
  • Use categories consistently for better reports
  • Always review the confirmation dialog before adding
  • Keep backups of your Excel file
  • Close the Excel file in other programs before using Ledgerly
  • Verify balance matches your expectations after adding transactions

Support & Troubleshooting

Common Issues

"Cannot save file - it may be open in another program"

Close the Excel file in Excel/Numbers/etc. Only Ledgerly should have the file open.

"Invalid number format"

Use numbers only (e.g., 45.50, not £45.50). Use decimal point, not comma (45.50, not 45,50). Maximum 2 decimal places.

"Failed to load Excel file"

Check that the file path is correct, you have read/write permissions, and the file isn't corrupted.

Graphs not showing

Graphs require an internet connection for the Plotly renderer. All data stays local; only the graphing library loads from CDN.
