User Guide

Learn how to manage your treasury effectively with Ledgerly.

Getting Started

1

First Launch

When you first run Ledgerly, you'll be asked to select or create an Excel file for storing your financial data.

  • Choose an existing file or create a new one
  • Store in OneDrive or iCloud for automatic backup
  • The location is savedâ€”you won't need to select it again
  • Ledgerly automatically creates necessary expense sheets

Adding Transactions

1

Select Transaction Type

Click either the EXPENSE button (red) or INCOME button (green). The selected button will be highlighted.

2

Fill in Details

Complete the transaction form with the following information:

  • Date: Auto-filled with today's date (YYYY-MM-DD format)
  • Description: What the transaction is for (required)
  • Amount: Enter in decimal format, e.g., 45.50 (required)
  • Category: Optional classification for reports
  • Notes: Optional additional information
3

Review and Confirm

Click "Add Transaction" to see a confirmation dialog showing transaction type, sheet name, amount, and description. This prevents accidental entries.

Viewing Your Data

1

Current Balance

Your balance is displayed prominently at the top right. Green indicates positive balance, red indicates deficit.

2

Recent Transactions

The main window shows your last 20 transactions with date, description, amount, type, and running balance. Switch sheets using the dropdown to view different transaction sets.

Generating Reports

1

Visual Reports

Ledgerly provides four types of interactive reports:

  • Expense Breakdown: Pie chart of spending by category
  • Balance Over Time: Line chart showing balance history
  • Monthly Summary: Bar chart comparing income vs expenses
  • Markdown Export: Professional tables organized by month

All charts open in your web browser and update automatically as you add transactions.

Best Practices

Enter Regularly

Add transactions daily or weekly to avoid backlog and maintain accurate records.

Use Categories

Consistent categorization makes your reports much more useful for analysis.

Review Confirmations

Always check the confirmation dialog carefully before clicking "Yes".

Backup Regularly

Keep backups of your Excel file. OneDrive and iCloud backup automatically.

Close Excel

Don't open the Excel file in other programs while using Ledgerly.

Verify Balance

After adding transactions, check the balance matches your expectations.

Excel File Format

Ledgerly manages your Excel file automatically. Each sheet contains these columns:

Column Description Format
Date Transaction date YYYY-MM-DD
Description What the transaction is for Text
Amount Transaction amount Decimal
Type Income or Expense Text
Category Optional classification Text
Notes Optional additional info Text
Balance Running balance (auto-calculated) Decimal
