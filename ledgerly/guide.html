User Guide

Learn how to use Ledgerly effectively.

Getting Started

1

First Launch

When you first run Ledgerly, you'll be asked to select or create an Excel file. This file will store all your financial data.

  • Choose an existing file or create a new one
  • Recommended: Store in OneDrive or iCloud for automatic backup
  • The location is saved‚Äîyou won't need to select it again
2

Automatic Sheet Creation

Ledgerly automatically creates the necessary expense sheets in your Excel file:

  • actual-to-date - Track real expenses and income
  • expected - Plan future budgets
  • Custom sheets - Create sheets for specific trips or categories

Adding Transactions

1

Select Transaction Type

Click either the EXPENSE button (red) or INCOME button (green). The selected button will be highlighted.

2

Fill in Transaction Details

  • Date: Auto-filled with today's date (YYYY-MM-DD format)
  • Description: What the transaction is for (required)
  • Amount: Enter the amount in decimal format, e.g., 45.50 (required)
  • Category: Optional classification for reports (e.g., "Travel", "Supplies")
  • Notes: Optional additional information
3

Review and Confirm

Click "Add Transaction" and a confirmation dialog will appear showing:

  • Transaction type (Income or Expense)
  • Sheet name
  • Amount and description

Click "Yes" to confirm or "No" to cancel. This prevents accidental entries.

Sheet Switching

Use the dropdown menu at the top left to switch between expense sheets. Each sheet maintains its own:

  • Transaction history
  • Running balance

Perfect for keeping different budgets separate (e.g., general funds vs. specific trips).

Best Practices

Enter Regularly

Add transactions daily or weekly to avoid backlog and keep records accurate.

Use Categories

Consistent categorization makes reports much more useful for analysis.

Review Confirmations

Always check the confirmation dialog before clicking "Yes" to prevent mistakes.

Backup Regularly

Keep backups of your Excel file. If stored in OneDrive/iCloud, backups are automatic.

Close Excel

Make sure the Excel file isn't open in another program while using Ledgerly.

Verify Balance

After adding transactions, verify the balance matches your expectations.

Understanding the Excel Format

Ledgerly manages your Excel file automatically. Each sheet contains these columns:

Column Description Format
Date Transaction date YYYY-MM-DD
Description What the transaction is for Text
Amount Transaction amount Decimal (positive number)
Type Income or Expense Text
Category Optional classification Text
Notes Optional additional info Text
Balance Running balance (auto-calculated) Decimal
